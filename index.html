Thoughts on Progessive Enhancement

When designing a webstie using graceful degradation, you design the website for specifically the best, most advanced websites. Then the older browsers are then ignored. Designing a webstie using progressive enchancement means starting with the most basic element of a wbesite, the content, and the adding the more compplex features, such as the Javascript. The reason this is important is it allows users on more techinoical limted browsers to enjoy all the content websites has to offer. It also bring sthe web experiince up for people on worse browsers.
